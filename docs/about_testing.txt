О тестировании.

1   См. \durak_game\unit_test - юнит тесты.

2   См. \durak_game\logic_test - тесты логики игры.

3   Для тестирования совместимости с сервером Виктора использовать написанный им тестер.
    Пример команды для запуска:
    java -jar foolTester.jar D:\dev\durak_by_victor\spec\durak\tests D:\dev\durak_by_victor\spec\durak\answers http://localhost/cgi-bin/commands.py 5
    Подробности об этой системе - в документации Виктора. Тесты, которые обязательно должны пройти, лежат в папке tests\compatib_tests

4   Тестирование ИИ. Для проверки работоспособности ИИ: запустить две копиии ИИ как противников и уйти пить чай, при необходимости - повторить.
    Качество работы можно проверить:
    1 Запуская его с ИИ с другим алгоритмом
    2 Играя с ним и анализируя его действия
