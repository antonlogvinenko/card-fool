Для установки требуется:

1 Прописать полный путь до папок client, server, common в файле durak.pth.
  Поместить этот файл в папку, куда установлен Python, например, C:\Python24.
  Теперь при выполнении кода дополнительно в указанных директориях Python будет искать модули.

2 Установить spyce. Установить из папки src\web\fool_client файлы для spyсе.
  Более подробно см. доки spyce.

3 Сделать .bat файл, запускающий spyce:
  spyce-2.1\spyceCmd.py -l
  Или несколько таких файлов, если нужно будет тестировать Несолько локальных клиентов:
  spyce-2.1\spyceCmd.py -l --conf "D:\dev\eclipse\durak_game\spyce-2.1\www\fool_client\config.py"
