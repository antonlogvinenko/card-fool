Для запуска тестирования логики игры необходимо:
1. Запустить сервер в тестовом режиме (server.py test).
2. Запустить run_logic_tests.py.

Ожидаемый результат. Если все тесты пройдены, в консоль будет выведен список
поддиректорий с тестами, после каждой поддиректории - пометка "...OK".
Иначе - соответствующее сообщение о несовпадении ответов сервера.

На выполнение всех тестов уходит не более 4 минут.
